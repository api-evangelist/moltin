# moltinThis is a repo for managing machine readable artifacts for the [moltin APIs](https://molt.in/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [moltin APIs](https://molt.in/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://molt.in/)- [Developer](https://molt.in/)- [Twitter](https://twitter.com/moltin)- [GitHub](https://github.com/moltin)The [APIs.json](https://github.com/api-evangelist/moltin/blob/master/apis.json) index provides links to all the relevant links for the moltin APIs, including the OpenAPI and Postman if they are present.
